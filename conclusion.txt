Conclusión de módulo.

1. ¿Qué se aprendió durante el desarrollo de proyecto?
Mientras llevabamos a cabo el desarrollo del programa, logramos aprender la sintaix que maneja Java. No osbtante, mientras se diseñaba el proyecto adquirimos un gran conocimiento a cerca 
de los contenedores de datos o mejor conocido como variables, cuya función es muy importante a la hora de almacenar información. Otro de lo conceptos aprendidos fueron los tipos primitivos, su clasificación y sus subtipos, cuyo aprendizaje facilita el reconocimiento de cada literal. La declaración de la funciones no se queda atrás, nos deja como enseñaza la descomposición funcional y cuál es la ventaja que tienen en cada programa para ejecutar la lógica ecrita dentro de ella, ya que al igual que las clases tienen una amplia importancia al momento de definir un objeto.

2. Una de las ventajas de haber tomado este módulo y haber puesto en práctica lo conceptos con el desarrollo de un programa es el debloqueo de una nueva habilidad que puede ser aplicada en la 
vida cotidiana para resolver problemas del diario vivir, o incluso para iniciar nuestro propio proyecto y darlo a conocer a personas que quieren resolver alguna necesidad. Esta experiencia puede 
sumarse como una habilidad a nuestro currículum independientemente del área en el que trabajemos, ya que puede ser un plus al momento de buscar un trabajo en la industria tecnológica. sin embargo, 
el haber desarollado este proyecto puede ser uno de los primeros pasos para iniciar una carrera como desarrollador, ya que de aquí podría nacer el gusto de aprender nuevos lenguajes, tecnologías o
incluso adentrarse totalmente en el mundo de Java.

3. Dificultades presentadas
Durante el desarrollo del proyecto, enfrentamos varias dificultades que requirieron tiempo y esfuerzo para superarlas. Una de las principales dificultades fue comprender y manejar adecuadamente las estructuras de datos necesarias para almacenar la información de los vendedores y productos, así como para calcular las ventas totales. La utilización efectiva de las clases Map para representar estos datos y la necesidad de realizar operaciones complejas con ellas resultaron ser un desafío significativo.

Además, tuvimos dificultades para garantizar la correcta lectura y procesamiento de los archivos de entrada. La manipulación de archivos en Java y la gestión de excepciones asociadas a la lectura y escritura de archivos que requeron un enfoque cuidadoso para evitar errores y asegurar la robustez del programa.

Otro desafío importante fue integrar la lógica de negocio en la clase Main.java para calcular las ventas totales y generar informes de manera eficiente. Esto implicaba diseñar algoritmos efectivos para recorrer y manipular los datos almacenados en las estructuras de datos, así como implementar la generación de archivos CSV con el formato correcto.

A pesar de estas dificultades, abordamos cada problema de manera sistemática y consultamos recursos adicionales, como documentación oficial de Java y tutoriales en línea, para comprender mejor los conceptos y encontrar soluciones viables.
